# Passwordchecker
This is the project i got, its basically designed to check the strength of a password.
Developed a command-line Password Strength Checker using Python.
-Implemented secure password input using the getpass module to prevent password visibility.
-Designed an algorithm to evaluate password strength based on:
-Lowercase characters
-Uppercase characters
-Digits
-Whitespace
-Special characters
-Used the string module for character classification and validation.
-Applied conditional logic to assign strength scores and generate security remarks.
-Implemented input validation and loop control for continuous user interaction.
